# uktobbackend

1) Activate the virtual environment:
  For Windows:
  ```
  env\Scripts\activate.bat.
  ```
  For Linux/macOS: 
  ```
  source env/bin/activate.
  ```
2) navigate to uktob directory
3) Run the command: 
   ```python app.py.```
4) Your Flask API will be accessible at http://localhost:5000.
